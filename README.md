# Bike Sales Customer Analytics Dashboard

An interactive Excel dashboard built to analyze customer purchase behavior from a 1,000+ record bike sales dataset — going from raw, inconsistent data to a clean, decision-ready dashboard.

## What it does
- Cleans and standardizes inconsistent categorical entries (marital status, region, occupation) using Find & Replace and formula-based corrections
- Segments customers into age brackets (Adolescent / Middle Age / Old) using nested IF formulas
- Uses Pivot Tables to analyze average income by gender and purchase counts by commute distance and age segment
- Combines bar and line charts with interactive slicers, letting users dynamically filter by customer segment to explore purchase-behavior trends

## Tools
Excel — Pivot Tables, nested IF formulas, Find & Replace, interactive slicers, chart design

## Why it matters
Real-world sales data is rarely clean. This project focuses as much on the data-cleaning and segmentation logic as on the final visuals, so the dashboard reflects trends that are actually accurate — not just good-looking.
